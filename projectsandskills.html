    Compiler desgin (C++)

I created a compiler capable of reading a unique language, in this case the language
established as FuegoDinero. This language is based on C++ with the expressions changed
in order for it to be unique. The purpose of this, is to be able to create a fully
working complier that is able to translate programs written in the FuegoDinero language
and run them, creating the desired output for it.

First, I needed to create the language, after that have the symbol table that later the
code generator would use to check the symbol table anytime it needed it.
After that worked on the scanner, to make sure that it scans for the tokens correctly,
which then led to code the parser, this parser uses the tokens and executes the code
in the desired language.
